Robotics Proyect

A robotics proyect that simulates the autonomous movement of a robot in a unsupervised enviroment. Using the ROS (Robotics Operating System) framework and some path planing algorithms like A* we were able to control the movement of the robot, the simulation was runned in the v-rep enviroment and the UI (User Interface) to saw the robot info and to indicate the destination point was builded using bootstrap, javascript and html.

Portable and Low-Cost Device for Audio Distress SignalRecognition in Urban Areas

Togheter with a machine learning group, we design a portable and low cost device for audiodistress signal recognition in real urban scenarios. We build a database of audio recorded in real urban areas using a low costmicrophone, using this database, we trained different architectures of convolutional neural networks that processed temporal and frequency features to accurately recognize audio distress signals in noisy environments. Then, we deployed the trained model on a Raspberry Pi.
